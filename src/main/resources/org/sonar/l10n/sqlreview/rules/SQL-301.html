Lambda selectAll() 等价于 SELECT *

使用 MyBatis Plus Lambda Wrapper 的 .selectAll() 方法会返回实体的所有字段，等价于 SQL 中的 SELECT *。

风险说明

selectAll() 的问题：

  • 返回不需要的列，浪费网络带宽
  • 增加内存消耗
  • 无法利用覆盖索引优化
  • 表结构变更可能导致程序错误

示例

// 错误
Wrappers.<AppUserEntity>lambdaQuery()
    .selectAll(AppUserEntity.class)
    .eq(AppUserEntity::getStatus, 1);

// 正确
Wrappers.<AppUserEntity>lambdaQuery()
    .select(AppUserEntity::getUserId, AppUserEntity::getUserName)
    .eq(AppUserEntity::getStatus, 1);

建议

  • 使用 .select() 方法明确指定需要的字段
  • 结合 DTO 模式只映射需要的字段
